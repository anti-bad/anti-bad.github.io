# Anti-BAD Challenge Website

## Quick Start

1. Install Jekyll:
   ```bash
   gem install jekyll bundler
   ```

2. Install dependencies:
   ```bash
   bundle install
   ```

3. Run locally:
   ```bash
   bundle exec jekyll serve
   ```

4. Open http://localhost:4000 in browser

### Build only (without serving):
```bash
bundle exec jekyll build
```
